# Regex ejercicios de practica

Este repositorio contiene una colección de ejercicios diseñados para ** demostrar, practicar y fortalecer mis habilidades en expresiones regulares (REGEX)**.

Los ejercicios fueron propuestos por ChatGPT con dos niveles de dificultad (media y avanzada), y **todos han sido resueltos completamente por mí**, como parte de un proceso de aprendizaje y autoevaluación.

---

## Objetivo del repositorio

- Mejorar el dominio de **expresiones regulares**.
- Practicar validaciones, extracción y transformación de texto.
- Comprender casos reales y complejos de uso de REGEX.
- Desarrollar soluciones claras, eficientes y mantenibles.

Este repositorio tiene fines **educativos y de práctica personal**.

---

## Niveles de dificultad

### Dificultad MEDIA (10 ejercicios)

1. Extraer todos los números de un texto y guardarlos en un `List<Integer>`.
2. Validar si una contraseña tiene mínimo 8 caracteres, una mayúscula y un número.
3. Contar cuántas palabras empiezan con mayúscula en un texto.
4. Extraer todas las fechas con formato `dd/mm/yyyy`.
5. Reemplazar todas las vocales por `*`.
6. Dividir un texto en palabras ignorando múltiples espacios.
7. Validar una matrícula con formato `ABC-123`.
8. Eliminar todos los caracteres especiales excepto espacios.
9. Extraer los dominios de una lista de correos electrónicos.
10. Contar cuántas veces aparece una palabra sin importar mayúsculas/minúsculas.

---

###  Dificultad AVANZADA (10 ejercicios)

1. Validar una contraseña fuerte (mayúscula, minúscula, número y símbolo).
2. Extraer etiquetas HTML sin capturar su contenido.
3. Detectar números decimales positivos y negativos.
4. Reemplazar palabras prohibidas por `***` sin afectar coincidencias parciales  
   (ejemplo: `mal` ≠ `malo`).
5. Validar direcciones IPv4.
6. Extraer valores entre comillas dobles.
7. Validar un JSON simple utilizando REGEX.
8. Detectar palabras duplicadas consecutivas.
9. Normalizar texto (acentos, espacios y mayúsculas).
10. Extraer funciones y sus parámetros desde código Java.

---

- Uso intensivo de:
  - `Pattern`
  - `Matcher`
  - Expresiones regulares avanzadas
